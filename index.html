🎯 Foco Diário: Organizador de Estudos CLI

✅ Online

📖 Sobre o Projeto

Aplicação CLI em Python que ajuda estudantes a manterem o foco limitando a no máximo 3 tarefas por dia. Combate a sobrecarga cognitiva e a procrastinação.

✨ Funcionalidades

  • 💡 Frase motivacional diária (via API ZenQuotes)
  • 📋 Adicionar tarefas (máximo 3)
  • ✅ Listar e concluir tarefas
  • 💾 Persistência em JSON

🚀 Como executar

git clone https://github.com/GAbrielSantMAc/foco-diario-estudos.git
cd foco-diario-estudos
pip install -r requirements.txt
python src/app.py
